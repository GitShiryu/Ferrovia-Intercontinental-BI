# MRS-Operational-Safety-Automation
"Pipeline de Monitoramento e Automação de Segurança: Detecção de anomalias em tempo real via Python e Integração de Logs (SIEM/Wazuh) para Infraestruturas de Missão Crítica.
Este projeto simula a coleta de logs de sistemas críticos (usando minha VPS como modelo) e automatiza a detecção de anomalias, enviando alertas em tempo real. É o tipo de lógica escalável que pretendo aplicar na Segurança Operacional da MRS.
